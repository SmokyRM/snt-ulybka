# Биллинг (начисления и итоги)

- Создайте период начислений (год/месяц) в /admin/billing.
- Проверьте начисления по участкам и итоговые суммы.
- Для электроэнергии используйте пересчёт по показаниям.
- Для членских/целевых применяйте «Начислить всем» (если доступно).
- Импортируйте платежи из CSV: /admin/billing/import.
- Проверьте журнал импортов: /admin/billing/imports.
- Контролируйте долги по участкам: /admin/debts.
