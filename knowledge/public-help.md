# Публичная помощь (жители СНТ)

- Вход доступен по коду участка, который выдаёт правление.
- Если кода нет, запросите его через личный кабинет: раздел «Доступ к участку».
- После входа заполните профиль (ФИО и телефон).
- Подтверждение членства выполняет правление — это занимает время.
- Документы СНТ доступны в разделе /documents.
- По вопросам оплаты смотрите реквизиты и сроки на странице /fees.
- По электроэнергии есть отдельная страница /electricity.
- Если участок уже привязан, обратитесь в правление.
- Контакты правления — в разделе /contacts.
- Подробная справка: /help
