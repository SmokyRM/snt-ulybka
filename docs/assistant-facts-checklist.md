## Проверки блока facts в ассистенте

- Авторизованный пользователь: спросить «какой у меня долг» → в ответе виден блок facts (статус и задолженность без сумм), текст ответа без цифр.
- Pending пользователь: спросить «что дальше» → статус pending в facts, инструктивный ответ.
- Guest: задать любой вопрос → facts не показываются.
- Smalltalk («привет») → ответ без facts.
- Оффтоп («как приготовить пасту») → нет facts и нет подсказок базы знаний/шаблонов.
