## Проверки страницы /cabinet/billing

- Авторизованный пользователь открывает /cabinet/billing → видит сводку статуса (verified/pending/…) и факт задолженности без сумм.
- Блок «Как оплатить» показывает пошаговые советы.
- Реквизиты: если заполнены — видны и копируются по кнопке; если нет — выводится «Реквизиты уточняются» + ссылка на знания.
- Блок «Если не согласны» содержит ссылки на шаблоны /cabinet/templates/claim и /cabinet/templates/appeal.
- Гость или админ без режима user-view перенаправляются (гость → /login, admin-view≠user → /admin).
