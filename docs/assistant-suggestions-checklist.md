## Проверки подсказок ассистента

- Smalltalk (\"привет\") → есть дружелюбный ответ, **без** блоков \"Материалы\"/\"Документы\".
- Навигация \"Как оплатить взносы?\" → в ответе есть блок \"Материалы\" (1–2 статьи) и \"Документы\" (если релевантно).
- Шаблонный вопрос \"Написать обращение\" → показывает кнопку на шаблон документа (гость → /templates/..., пользователь → /cabinet/templates/...).
- Слабый поиск (\"реквизиты\") → подсказки появляются только при score ≥ 3, не заваливают нерелевантными.
- Оффтоп (\"как приготовить пасту\") → ответ без suggestions (они скрыты при низком score).
- Ошибки/500 → ассистент отвечает или показывает техническую ошибку, чат остаётся на текущей вкладке.
