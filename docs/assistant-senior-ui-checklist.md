## AssistantWidget (упрощённый UI)

- Вкладки переименованы: «Быстрые ответы», «Задать вопрос», «Связаться», под ними есть краткое описание активной вкладки.
- При первом открытии виден подсказочный блок и исчезает после «Понятно» (сохраняется в localStorage), повторно не показывается.
- Быстрые чипы: сразу не больше 4, остальные только после «Ещё темы».
- Блоки «Материалы»/«Документы» свёрнуты по умолчанию, кнопки «Подробнее (N материалов)» / «Документы (N шаблонов)» раскрывают до 2 ссылок.
- При клике по теме/примеру во время загрузки видно «Открываю ответ по теме…», в ответе есть строка «По теме: …».
- Уточняющий блок появляется только при пустом/очень коротком ответе или явной просьбе уточнить, не для smalltalk.
- Под каждым ответом (кроме smalltalk) есть CTA «Не нашли ответ? Связаться», открывающий вкладку «Связаться».
- Текст/кнопки крупнее (16–18px), элементы кликабельны с высотой ≥44px, длинные ответы сворачиваются и раскрываются по кнопке.
