## Assistant Widget — масштаб UI (ручка в левом верхнем углу)

- При ширине > 480px видна ручка масштаба (иконка ⇆), курсор ew-resize, тянется влево/вправо.
- При ширине < 480px ручка скрыта; другие элементы (например, «Аа») остаются.
- Тяни вправо → интерфейс становится крупнее, влево → мельче; диапазон от 0.90 до 1.25.
- Масштаб сохраняется в localStorage (assistantUiScale:v1) и восстанавливается при следующем открытии.
- Масштаб влияет на внутренний контент, но не ломает размер окна/ресайз-ручку.
- При закрытии/сворачивании виджета ручка не мешает, восстановление масштаба работает.
