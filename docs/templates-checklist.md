## Templates MVP checks

- `/templates` открывается; видны шаблоны, работают поиск и фильтр по категории.
- `/templates/[slug]` открывается; отображаются заголовок, дата, текст с плейсхолдерами.
- Кнопка «Скопировать» копирует текст и показывает статус «Скопировано».
- Плейсхолдеры подставляются как `________`, дата — текущая.
- `npm run lint` и `npm run typecheck` проходят.
